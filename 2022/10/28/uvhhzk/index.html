Bing搜索引擎收录网站

发表于2022-10-28|更新于2022-10-28|教程Hexo
|字数总计:374|阅读时长:1分钟|阅读量:

添加网站
先访问 必应网站管理员工具，自行登录注册，然后点击左上角的添加网站。

在页面中填入你的网站域名，点击添加。

验证网站

添加网站后，我们通过 XML 文件的方式验证网站，只需要将该文件上传到网站的根目录即可。
注意：hexo 博客可能会编译该文件，编译后手动在 github 修改一下即可。XML 文件要放在 hexo 源码中的 source 文件夹内

上传完毕后点击按钮验证即可。看到该界面即代表添加成功。

配置网站地图

接下来我们用在线生成器给网站生成 sitemap.xml 文件，该文件指向搜索引擎爬虫可索引的链接。
这里推荐一个工具：https://www.xml-sitemaps.com/
进入网站后输入你的网站域名，点击 START。

然后点击下方按钮查看详细信息。

点击下载 sitemap.xml 文件。

然后将该文件上传到 hexo 源码中的 source 文件夹内，和上面的 XML 文件放置位置一样。
接下来我们在 Bing 中提交网站地图。输入 域名/sitemap.xml，并点击提交。

然后下方显示正在处理就代表配置完成啦。

提交完成后就可以等待 Bing 收录你的网站啦，一般收录需要半个月到一个月，可能会出现两个月的情况，请耐心等待。
